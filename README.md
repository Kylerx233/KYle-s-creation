# 《江山千里——绘梦成型》

## 项目简介
《江山千里——绘梦成型》是一个基于 Python、MediaPipe 和生成式 AI 的中国山水数字绘画交互系统。
用户可以通过手绘或手势输入生成草图，系统将结合 AI 将草图转化为具有《千里江山图》青绿山水意境的数字绘画，并在后续加入粒子互动效果。

## 技术栈
- Python
- PyQt6
- OpenCV
- MediaPipe
- Pillow
- NumPy
- Requests
- python-dotenv

## 开发计划
- 搭建项目工程结构与基础界面
- 集成手势识别与画布交互
- 接入 AI 图像生成能力
- 增加粒子互动与视觉效果优化
